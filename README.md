avito-map
=========

Search Avito with map


## Installation ##

Just run:
    $ npm install [--msvs-version=201*]

## Usage ##

Run it before import:
    $ mongo
    > use avito
    > db.test.clear()

Import some advert from avito.ru:
    $ node import
    
Start web application:
    $ node www

Look at http://localhost:3000/
